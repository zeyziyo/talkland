# Git 커밋 워크플로우

## 1. 변경 사항 확인
```bash
git status
```

## 2. 변경 내용 상세 보기
```bash
git diff --stat
```

## 3. 모든 변경 사항 스테이징
```bash
git add -A
```

## 4. 커밋 (메시지는 상황에 맞게 변경)
```bash
git commit -m "feat: 기능 설명"
```

## 5. 원격 저장소에 푸시
```bash
git push origin main
```
